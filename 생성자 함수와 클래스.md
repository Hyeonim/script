● 생성자 함수 : 객체마다 반복되는 프로퍼티와 메서드가 있다면 객체 틀을 미리 정의해 놓고 필요할 때 마다 그 틀을 사용해서 객체를 만들 수 있다. <br>
               : 객체의 틀을 만들 때 사용하는 함수를 '생성자 함수'라고 한다.<br>
               : 생성자 함수를 직어 내는 객체를 '인스턴스' or '인스턴스 객체'라고 한다. <br>

```

this.메서드2 : function(){...},
...

}
```
