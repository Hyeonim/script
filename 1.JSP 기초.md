● 자바스크립트의 특징
: 모든 웹 브라우저에서 작동
: 웹 브라우저에서 실행 결과를 즉시 확인할 수 있다
: 풀스택 웹 개발 뿐아니라 다양한 용도의 프로그램을 만들 수 있음
: 다양한 자바스크립트 공개 API 사용 가능
: 다양한 라이브러리와 프레임워크를 사용 가능


● 입력과 출력 방법
alert() 함수 : 알림 창 표시
             : alert() 함수의 괄호 안에 메시지를 입력하거나 변수를 사용하여 알림 창에서 텍스트 or 변수값 표시
             기본형 : alert(내용)

confirm() 함수 : 확인 창 표시
               : [확인], [취소] 버튼이 있어 어떤 버튼을 클릭했는가에 따라 다르게 동작 가능
               기본형 : confirm(내용)


prompt() 함수 : 프롬프트 창에서 사용자가 간단한 값을 입력할 수 있는 창
              : 프로그램 실행에 필요한 값을 받을 때 사용
              기본형 : prompt(내용) or prompt(내용,기본값)

console.log() 함수 : 콘솔 창에 괄호 안의 내용을 표시
                   : 보통 중간중간 프로그램이 제대로 작동하는지 확인하는 용도로 쓰음
                   기본형 : console.log(내용) 

document.write() 함수 : 괄호 안의 내용을 웹 브라우저 화면에 표시
                      : 실제 웹 브라우저 화면에 내용을 표시할때에는 DOM이용
                      기본형 : document.write(내용)

● 변수란?
변수 : 프로그램에서 사용하기 위해 값을 담아놓는 바구니
     : 일반적으로 변수는 프로그램 안에서 값이 달라질 수 있는 데이터를 가리킴
     : but 프로그램 안에서 계속 값이 바뀌지 않더라고 변수로 만들어서 사용하는데 이것을 상수 변수라고 함
     : 변수의 이름을 지정할 때 같은 이름을 사용하지 않아야함
     : var,let,const을 이용하여 사용 가능하고 var는 재선언과 재할당이 가능하여 사용하지 않음
     기본형 : let 변수명;, const 변수명;
            : let 변수명 = 값 or 식, const 변수명 = 값 or 식

● 자료형이란?
: 프로그램에서 처리하는 자료의 형태   (예) 3을 숫자로 처리할지, 문자로 처리할지
: 자료형, 자료 유형, 데이터 타입 등으로 부름
: 자바스크립트 자료형은 크게 ‘원시형’과 ‘객체’로 나눔
                             : 원시형(primitive type) : 하나의 값만 가지고 있는 자료형. 
                             : 객체(object): 원시형 외의 모든 자료

number(숫자) : 따옴표 없이 표기한 숫자
string(문자열) : '이나 "으로 묶어 나타냄
boolean(논리형) : true or false 두 가지 값만 가지고 있음
undefined : 자료형을 지정하지 않았을 때의 유형, 변수를 선언만 하고 값을 정의하지 않으면 undefined가 됨
null : 값이 유효하지 않을 때의 유형

● typeof() 함수 : typeof(값 또는 변수)를 넣어서 어떤 자료형인지 알려줌

● 배열과 인덱스
배열 : 하나의 변수를 여러 값을 할당할 수 있는 형태로 []안에 묶고 각 값을 ,료 구분
    기본형 : 배열명 = [값1,값2,값3]

인덱스 : 배열에 있는 여러 값을 저장하는 번호로 인덱스는 0부터 시작
